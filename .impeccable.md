## Design Context

### Users
Middle management and newly promoted leaders. They value clarity, efficiency, and tools that help them make complex decisions under pressure.

### Brand Personality
**Precise, Technical, Playful**
The interface feels like a professional engineering or tactical tool, but with a "game-like" reactivity that rewards exploration.

### Aesthetic Direction
**Tactical Command Center**
Dark backgrounds (OLED-tinted), illuminated data points, high-contrast states for instability, and a layout that prioritizes situational awareness.

### Design Principles (Impeccable)
1. **Atmospheric Depth**: A deep, neutral-tinted dark mode base using OKLCH to create a professional, low-light "command center" environment.
2. **Technical Layout**: Use monospace elements for data-heavy sections and precise border treatments to imply accuracy.
3. **Heroic Asymmetry**: The Shared Model board is the center of gravity. It should dominate the viewport, with status gauges and player controls positioned asymmetrically to create visual tension and flow.
4. **Instability through Light**: Use glow and vibration (lightly expressed) for "drifting" or "collapsed" fields to demand immediate attention.
5. **Fluid Authority**: Fluid typography and spacing systems based on a 4pt grid, ensuring the complex board is always at maximum legibility.

### The One Big Thing
**The Shared Model Board**
Every UI element exists to support the player's understanding of the 13-field model.
